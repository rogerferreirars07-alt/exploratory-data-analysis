# Exploratory Data Analysis

This repository contains exploratory data analysis (EDA) projects developed using Python.
The main goal is to demonstrate practical skills in data cleaning, data exploration,
data visualization and insight generation using real-world datasets.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## Repository Structure
Each notebook focuses on a specific dataset and analysis objective.
